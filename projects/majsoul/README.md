# 다음 한 수 — Shanten Lab

작혼(Mahjong Soul) 화면을 캡처해 추천 타패를 오버레이로 띄우는 안드로이드 도우미
`majsoul-helper-android`(비공개)의 심장 — 샹텐 DFS와 유효패(ukeire) 계산 — 을
Java에서 JS로 함수 단위 포팅해 브라우저에서 돌리는 데모. **의존성 0**.

**Live:** https://lamgul.github.io/projects/majsoul/

## 데모가 하는 일

- **손패 14장을 유니코드 마작패로.** 패를 클릭하면 교체 팔레트가 열려 직접 편집.
- **프리셋 4종.** 텐파이 직전(원본 레포 단위 테스트의 손패 그대로) / 치또이 노림 /
  국사 노림 / 애매한 중반.
- **바뀔 때마다 실계산.** 현재 샹텐(일반형 DFS · 치또이 · 국사),
  루트별(최단 화료/일반형/탕야오/치또이/국사) 추천 타패 카드,
  버림패 후보별 샹텐·유효패 표 — 전부 포팅한 엔진이 그 자리에서 계산하고
  소요 ms를 표시한다. 가짜 결과 없음.

## 원본과 다른 점

- 화면 캡처·타일 분류기(MediaProjection, `tile_templates.bin`)는 생략 —
  브라우저엔 읽을 화면이 없으니 손패를 직접 편집한다.
- 도라·버림패·후로 입력이 없어 남은 장수를 손패만 보고 센다 — 유효패가 실제 앱보다 후하게 잡힌다.
- 포팅하며 보탠 건 분석 1회 동안의 DFS 메모이제이션 하나.

`majsoul.js` 한 파일이 엔진과 UI 전부다.
