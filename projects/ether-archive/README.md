# 에테르 아카이브 — Ether Archive

핵겨울의 서울, AI가 만든 가짜 낙원. 30종 생존 역할 · 27종 능력 · 25종 엔딩을 가진
한국어 텍스트 어드벤처의 **플레이어블 슬라이스**.

**Live:** https://lamgul.github.io/projects/ether-archive/

원작은 비공개 레포(lamgul/ether-archive)의 단일 HTML 브라우저 게임이다.
게임 본편과 세계관·시스템 문서, 1부~10부 장편 원고, Node 자동 플레이테스트 20여 종이
한 레포에 사는 1인 게임 스튜디오 모노레포. 이 페이지는 그 코어 루프만 떼어 재현한다.

## 데모가 재현하는 것

- **역할 선택.** 원작 `JOBS` 데이터에서 4종(야전 의무병·고고학자·정보 브로커·전직 군인)을
  이름·시작 능력·시작 물품 그대로 이식, 회차마다 3종을 섞어 제시.
  공통 지급품(방진마스크·부모의 찢어진 사진·낡은 VR 장비(파손)) 규칙도 원작 그대로.
- **3칸 HUD.** 체력/정신/배고픔 ▣▣▣ — 어느 하나가 0이 되면 회차가 끝난다(영구죽음).
- **게이트 선택지.** `(관찰 Lv.1)` 능력 게이트와 `응급 붕대 1 소모` 아이템 게이트,
  미충족 시 회색 잠금 — 원작 표기 규칙을 따른다.
- **확정 vs 도박.** 성공 확률이 표시되는 도박 선택과 '방금 선택의 여파' 인라인 결과.
- **엔딩 3종 + 사망.** 원작 25종 중 세 계열(기록·생존·접속)의 축약판.

씬 일러스트는 원작의 흑백 손그림 PNG 대신 같은 톤의 인라인 SVG 플레이스홀더.
`ether-archive.js` 한 파일, 의존성 0, 네트워크 요청 0.
