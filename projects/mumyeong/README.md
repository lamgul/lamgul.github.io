# 무명 — Mumyeong Series

이름을 부르면 병드는 서울(무명록), 검은 비가 기억을 지우는 밤(무명비: 영만의 밤).
같은 세계관 공식으로 만든 두 텍스트 어드벤처와 그 제작 파이프라인을 소개하는 페이지.

**Live:** https://lamgul.github.io/projects/mumyeong/

원본 세 레포(`mumyeongrok`, `mumyeongbi-yeongman-ui-bam`, `common-worldbuilding-assets`)는
비공개다. 이 페이지의 데모는 그 레포에서 발췌한 실제 데이터로 핵심을 다시 세운 재현이다.

## 데모가 보여주는 것

- **두 게임 카드.** 스탯 구조(무명록: 몸/정신/공복/호명 0~4 미터, 무명비: 박스 게이지 +
  기억도·인간성·이름 부채)와 엔딩 수(11종 vs 14종)를 나란히 비교한다.
  엔딩 이름은 원작 데이터 그대로다.
- **무명록 프롤로그 요약판.** 원작 `게임.js`의 `SCENES` 테이블에서 막차→역→마포 5장면을
  발췌했다. 스탯 미터, 단서 칩, 조건부 잠금 선택지, 호명 경고 연출, 호명 3에서 끼어드는
  E7 '세 번째 호명' 위기 분기까지 원작 규칙을 따른다.
- **세계관 생성기.** 공용 저장소의 `소재_라이브러리.md`에 있는 장소 10, 이상 현상 9,
  생활 소품 10, 세력 9, 금기 8 — 46개 요소를 그대로 이식해, "장소 1 + 이상 현상 1 +
  생활 소품 1 + 세력 1 + 금기 1" 변주 공식으로 새 게임 컨셉을 뽑는다. 요소별 재추첨 가능.

## 재현하지 않은 것

localStorage 저장과 기록실, 인카운터 풀 대부분, 관계(신뢰)·능력 시스템, E7 외의 엔딩 분기.
캐릭터 일러스트 64장은 원천 자산을 비공개로 둔다는 저장소 원칙에 따라 싣지 않았다.

## 파일

```
index.html    페이지 골격 (공용 base.css / project.css 위에 얹힘)
mumyeong.css  데모 무대 전용 스타일 — 원작이 다크 터미널 UI라 무대만 늘 어둡다
mumyeong.js   데모 전체 — 카드 · 요약판 상태 머신 · 생성기, 의존성 0
```
